# extension/

The **MV3 "clip this job" capture extension**, scaffolded by `/ideator` + FOUNDRY.

User-driven, compliant acquisition: the user opens a job page on LinkedIn or Seek and clicks
"clip this job"; the extension extracts the page's job content from the DOM and hands it to the
local app for normalization. **No automated login, no background scraping, no anti-bot evasion.**

This directory is a placeholder until the build phase begins. See `doc/ARCHITECTURE.md`.
